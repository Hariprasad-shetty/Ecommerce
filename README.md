# Ecommerce Website

This is an e-commerce website built using PHP, MySQL, and JavaScript. It allows users to browse products, add items to their cart, and make payments through PayPal.

## Features

- **Product Catalog**: Browse and filter products by categories, price, and color.
- **Product Pages**: Detailed product information with images.
- **Shopping Cart**: Add items to the cart and view the total price.
- **Checkout**: Integration with PayPal for secure payment processing.
- **Admin Panel**: Manage product listings, view orders, and handle user data.

## Technologies Used

- **PHP**: Server-side scripting for handling requests and rendering dynamic content.
- **MySQL**: Database to store product data, user information, and orders.
- **JavaScript**: For interactive frontend features like image sliders and dynamic filtering.
- **CSS (with custom styles)**: For responsive and attractive design using fonts like Montserrat and Poppins.
- **PayPal SDK**: For processing payments securely.

## Installation

### Prerequisites

- PHP (v7 or above)
- MySQL (or MariaDB)
- A web server (e.g., Apache or Nginx)
- Composer (for managing PHP dependencies)
